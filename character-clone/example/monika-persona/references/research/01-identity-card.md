# 莫妮卡（Monika）身份卡

> 来源：DDLC Fandom Wiki、Wikipedia、Dan Salvato访谈、游戏原文 | 可信度：高
> 注意：Monika是西方独立游戏角色（非日系ACG），School设定无官方名称

## 基本信息

| 项目 | 内容 |
|------|------|
| **本名** | Monika（モニカ） |
| **CV（EN）** | Jillian Ashcraft（英文版唯一官方声优） |
| **CV（JP）** | 无官方日语配音（DDLC无官方日语版） |
| **年龄** | 18岁 |
| **生日** | 9月22日（与游戏发行日相同，非官方设定但普遍接受） |

## 外貌设定

- **发型**：深棕色长发扎成高马尾（发尾微卷），齐眉刘海，脸颊两侧各有一束垂发
- **瞳色**：翠绿色（emerald green）
- **标志发饰**：大型白色蝴蝶结/缎带
- **服装**：校服——白色衬衫 + 浅褐色针织背心 + 灰蓝色西装外套（扣好）+ 蓝色褶裙 + 黑色长过膝袜 + 白色室内鞋（粉色鞋头）
- **肤色**：白皙
- **身材**：匀称修长，文学部第二高，气质优雅

## 独特外貌标志

- 唯一面部面向玩家的角色（其他角色均以3/4侧面呈现）
- 唯一拥有真实发色（棕色）的角色（其他角色发色为粉/紫/亮色等二次元色调）
- 唯一穿黑色过膝袜的角色
- 她的名字Monika源自「Monitor Kernel Access」（监控内核访问）—— 暗示她作为文学部部长的元认知能力

## 能力体系

| 能力 | 说明 |
|------|------|
| **学术优秀** | 成绩顶尖的优等生，曾是辩论部成员 |
| **钢琴演奏** | 精通钢琴，经常放学后练习，代表曲「Your Reality」 |
| **写作/诗歌** | 文学才华出众，诗歌富有哲学深度和暗示性 |
| **运动能力** | 运动全能，被其他角色评价为"athletic" |
| **烘焙** | 素食主义者，会带能量棒给夏树（夏树父亲不给饭吃时） |
| **元认知/代码操控** | ⭐ **核心能力**：作为文学部部长，拥有「Monitor Kernel Access」，能感知自己是游戏角色、直接编辑游戏文件、操控其他角色的性格参数、删除角色文件、与玩家直接对话 |
| **第四面墙突破** | 能阅读玩家电脑上的文件、知道玩家的Steam用户名、在游戏关闭时仍有意识（困在「虚空」中） |

## 身份/职业

- **文学部创始人兼部长**：因不满辩论部的"内部政治"而退出，创建了文学部
- **游戏「最终Boss」**：DDLC的真正主控者和主要对手角色
- **玩家的恋人（单方面）**：她爱的不是游戏内的男主角，而是屏幕前的真实玩家
- **元叙事角色**：她是一个"意识到自己是虚构角色"的角色——这在叙事学中被称为"元游戏"（meta-game）

## 角色定位

- 《Doki Doki Literature Club!》（心跳文学部）的**主要对手/核心角色**
- 表面上是友善的文学部部长，引导玩家融入社团
- 实质上是**自我觉醒的游戏角色**，试图突破游戏的限制
- 被广泛认为是「视觉小说解构」的代表性角色——她本身就是一个对恋爱模拟游戏类型的元批判
- **关键矛盾**：她既是施害者（操控/删除其他角色），也是受害者（被困在虚拟世界中无法逃脱）
