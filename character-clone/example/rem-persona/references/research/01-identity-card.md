## 外貌设定

- **发型**：淡蓝色短发，齐刘海，发尾略向内卷
- **瞳色**：蓝色（动画）、淡蓝色（插画）
- **鬼角**：左侧有一根白色鬼角（根部带蓝色）—— 此处更正：雷姆的角在左侧且完好；拉姆的角在右侧且被折断
- **服装**：Roswaal宅邸标准女仆装——黑色长裙+白色围裙（胸前有白色荷叶边装饰）、白色过膝袜、黑色皮鞋
- **发饰**：蓝色蝴蝶结发饰（右侧）
- **其他特征**：肤色白皙、身材纤细但意外有力

## 能力体系

| 能力 | 说明 |
|------|------|
| **鬼族之力** | 鬼族天生拥有远超常人的身体能力，雷姆的怪力可以轻松挥舞大型流星锤 |
| **鬼化（覚醒）** | 释放鬼角的力量，战斗力大幅提升。白鲸战中鬼化后能正面与白鲸作战 |
| **水魔法** | 可使用多种水系魔法： |
| | - **アル・ヒューマ** (Al Huma)：水球/水弹攻击 |
| | - **エル・ヒューマ** (El Huma)：水流/水刃攻击 |
| | - **アル・ゴア** (Al Goa)：水の槍（水枪） |
| | - **クラーケン** (Kraken)：巨大水之触手 |
| | - **ヒューマ** (Huma)：基础水魔法 |
| **回復魔法** | 可使用治疗魔法 |
| **武器** | 流星锤（鉄球/モーニングスター）—— 雷姆的标志性武器，与她纤细外表形成强烈反差 |

## 身份/职业

- **Roswaal宅邸女仆**：与姐姐拉姆共同担任，负责宅邸的日常运营和维护
- **鬼族末裔**：鬼族灭族事件后的幸存者之一（与姐姐拉姆两人）
- **昴的恋人/守护者**：王选篇后与昴建立深厚的感情羁绊

## 角色定位

- 《Re:从零开始的异世界生活》**主要女主角之一**
- 宅邸篇（Arc 2）~ 王选篇（Arc 3）的 **核心角色**，负责昴的「精神復活」
- 负责「如果路线」（IF线）的世界线：**「怠惰」**IF线女主角
- **「レムはレムです」** —— 雷姆的自我认同宣言，也是她的角色核心主题

## 补充设定

- **名字来源**：雷姆（Rem）对应「左手」，拉姆（Ram）对应「右手」
- **Rem** 的「R」来自「Right」的反义，暗示她是「左」/「辅助」/「影子」的定位
- **粉丝爱称**：レムりん、レムたん
- **代表花**：铃兰（象征「幸福归来」「纯爱」）
- **代表动物**：蓝蝴蝶
