# Vistual-Charater-clone/角色造灵
A Skill for cloning virtual characters by collecting information from relevant websites to generate 6-dimension character profiles, refined through scoring to produce a character_personality.md file.
Warning: This skill consumes a large amount of tokens. Please be mindful of usage costs. Generating one character costs approximately: 1-2 CNY (Deepseek-v4-flash).
Inspiration: huashu-nuwa
Usage Guide:
1.Trigger words: clone XX, make a character, character distillation, virtual character, anime character, game character, roleplay, character clone, create    a persona for XX, mold a character
  Also triggers on vague requests: I want to roleplay a character, Is there a character suitable for me to play?, Help me design a character persona

2.Confirm the character requirements

3.Create directory structure and begin research, write the file upon completion

4.Confirm the character framework, perform dual-agent 4-dimension similarity scoring, and finally generate the character document
